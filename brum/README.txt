Antes de utilizar se deben actualizar todas las rutas de los archivos manualmente. Por defecto utiliza la ruta "http://localhost/brum/". En un futuro se planea mover todas las rutas a un archivo comun para asignarlas a los enlaces.
Tambien debe editar el archivo "dbConn.php" para que se conecte con su base de datos.
Por ultimo tendra que ejecutar el script sql antes de utilizar el software.
